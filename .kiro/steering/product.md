# Product Overview

株式会社テックリード (TechLead Inc.) のコーポレートホームページ。企業情報、サービス紹介、実績紹介、採用情報、お問い合わせ機能を提供する。

## Core Capabilities

- **企業情報の発信**: 会社概要、理念、技術スタック、開発プロセスの公開
- **サービス・実績の紹介**: 提供サービスと実案件の詳細を構造化データで管理
- **お問い合わせ受付**: フォームバリデーション、メール通知機能
- **採用情報の提供**: 募集職種とポジション詳細の掲載

## Target Use Cases

- **企業ブランディング**: 技術力と開発プロセスの透明性をアピール
- **リード獲得**: お問い合わせフォームによる見込み客の獲得
- **採用活動**: エンジニア採用のための情報発信

## Value Proposition

**データドリブンコンテンツモデル**: 全コンテンツを TypeScript で型安全に管理し、プレゼンテーションと完全分離。`web/src/data/` 配下にビジネスロジックを集約することで、非エンジニアでも構造化されたデータ編集が可能。

**フルスタック型の構成**: フロントエンド (React SPA) とバックエンド (Cloudflare Worker) をモノレポで統合し、共通スキーマ (`@homepage/shared`) により一貫したバリデーションを実現。
